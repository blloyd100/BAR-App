# BAR-App

**Purpose:** Help people destress through an app using different techniques

**Techniques:** 
* Nature sounds 
* Daily check-ins 
* Meditation
* White noise
* Calming pictures
* Breathing exercises

**Resources:**
* Phone a psychologist
* Therapist based on location
* Feedback based on answers to series of questions
